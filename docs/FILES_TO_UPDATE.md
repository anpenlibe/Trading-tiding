# 📋 Files Requiring Regular Updates

## Daily Updates
| File | Update Frequency | What to Update |
|------|------------------|----------------|
| `PROJECT_TOC.md` | Daily | Progress, new files, status changes |
| `docs/decisions_day_XX.md` | Daily | New decisions made |
| `.env` | As needed | API keys, settings |
| `data/logs/*.log` | Automatic | Generated by system |

## Weekly Updates
| File | Update Frequency | What to Update |
|------|------------------|----------------|
| `README.md` | Weekly | Major features, setup changes |
| `src/README.md` | When modules change | New modules, API changes |
| `requirements.txt` | When adding packages | New dependencies |
| `CHANGELOG.md` | Weekly | Version history |

## As-Needed Updates
| File | When to Update | What to Update |
|------|----------------|----------------|
| `src/config.py` | Config changes | Trading parameters, symbols |
| `docs/architecture/*.md` | Design changes | System design updates |
| API documentation | API changes | Endpoints, parameters |
| Test files | Feature changes | New test cases |

## Git Commit Message Format
```
feat: Add new feature
fix: Fix bug
docs: Update documentation
refactor: Refactor code
test: Add tests
chore: Update dependencies
```
