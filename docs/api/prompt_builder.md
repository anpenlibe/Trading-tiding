# prompt_builder

AI prompt construction utilities.

## Class: `PromptBuilder`

Build structured prompts for Claude AI.

### Methods

#### `safe_format(value, default, decimals)`

Safely format a value that might be None.

#### `create_analysis_prompt(symbol, market_data, indicators, context)`

Create a structured analysis prompt for Claude.

#### `parse_response(response_text, current_price)`

Parse Claude's response into structured data.

#### `create_context(strategy, capital, risk_level)`

Create trading context for prompt.
