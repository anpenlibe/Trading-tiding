# monitor

Alert monitoring dashboard.

## Functions

### `display_alert_status(alert_engine)`

Display current alert status.

### `display_alert_rules(alert_engine, symbol)`

Display detailed alert rules information.

### `display_alert_history(triggered_alerts)`

Display recent alert history.

### `get_alert_summary(alert_engine)`

Get alert system summary statistics.

### `monitor_alerts_realtime(alert_engine, data_source_func)`

Real-time alert monitoring (for testing/debugging).

### `create_alert_dashboard()`

Create a simple text-based alert dashboard.
